## Steps to Run BlockChain on NodeJs
  - Step 1: Run src/KeyGenerator.js & Copy Public & Private key on Notepad
  - Step 2: Open src/main.js Replace SenderPrivateKey & ReceiverPublicKey
  - Step 3: Run src/main.js
